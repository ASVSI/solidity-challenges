# ASVSI Security Challenge Proposals

Welcome to the **ASVSI Community Challenge Proposal System**!

This folder is dedicated to **community-suggested security challenges**. Contributors can propose new challenges, which may later be added to the official `challenges/` section if approved.

---

## How it works

1. **Fork the repository** and create a new branch.
2. Copy the `template.md` file and fill in your challenge proposal.
3. Submit a **Pull Request (PR)** from your branch to `main`.
4. The ASVSI team will review your proposal based on:

   * Alignment with existing categories
   * Difficulty level (Beginner / Intermediate / Advanced)
   * Practicality and educational value
   * Avoiding duplicates
5. Community members can **react with üëç** on your PR to show interest.
6. Approved proposals will be **moved to `/challenges`** and the contributor credited.

---

## Submitting a Proposal

1. Copy `template.md` into your branch.
2. Fill in all sections.
3. Submit a PR titled: `Proposal: Challenge Title`.
4. Tag any collaborators if relevant.

After review, approved proposals will receive a merge and be credited in the **ASVSI Contributors List**.

---

## Benefits for contributors

* Get your GitHub username listed as a **challenge proposer**.
* Contribute to an open-source security learning platform.
* Help beginners and peers engage with real security challenges.
* Gain recognition in the **ASVSI community and leaderboard**.
